# Code-Alpha-Task-1-Foody-Express-Wireframing-Low-Fidelity-Design

**Figma Link:** https://www.figma.com/design/GKuSZV4nIeSKN64zSUiB2j/Food-Delivery-App-WireFrames?t=namyf6TFDKZ4yEzR-1

**Project Overview:** Foody Express is a low-fidelity wireframing project for a modern food delivery mobile application. The project focuses on designing a seamless and intuitive user experience that enables users to discover restaurants, browse menus, place orders, and track deliveries efficiently. The wireframes were created to establish the application's information architecture, screen hierarchy, and navigation flow before moving to high-fidelity UI design. The design emphasizes simplicity, accessibility, and a streamlined ordering process to ensure a smooth user journey from login to order confirmation.

**Tools Used**
1. **Figma** – Used to create the low-fidelity wireframes and design the overall user flow for the Foody Express mobile application.
2. **Microsoft Word** – Used for preparing project documentation and writing design descriptions.
3. **PDF** – Used to export and share the final project report and wireframe documentation in a universally accessible format.

**Key Features**
1. User authentication and registration
2. Restaurant discovery and search
3. Menu browsing and item selection
4. Shopping cart management
5. Secure checkout process
6. Order confirmation and tracking

**Design Process**: The wireframes were developed by mapping the complete user journey and identifying the essential screens required for a food delivery platform. Each screen was designed with a focus on usability, minimizing user effort while maintaining clear navigation and functionality.

**Screens Included**
1. Login & Registration
2. Home Dashboard
3. Restaurant Menu
4. Cart Summary
5. Checkout
6. Order Confirmation

**Goal:** The goal of this project is to create a strong foundation for a scalable and user-friendly food delivery application that can later be transformed into an interactive prototype and high-fidelity design.
